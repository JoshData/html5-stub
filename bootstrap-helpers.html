AJAX Loading Indicator

Use my global function ajax_with_indicator in place of $.ajax to wrap the call with a loading indicator when the query might take a long time to return. It also shows a generic error message when the AJAX call fails (either with a failure status code or if an object like { 'status': 'error', 'message': 'your message here' } is returned).

ajax_with_indicator({ usual jquery ajax options })

Try it »
